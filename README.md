# AM/FM Modulator with Carrier Signal Generator & Audio Player

A Python-based communication engineering project that simulates Amplitude Modulation (AM) and Frequency Modulation (FM). The application generates a high-frequency carrier wave, modulates it using an audio signal, and includes a built-in audio playback system to visualize and hear the results.

---

## 🚀 Features

* *Carrier Signal Generation:* Dynamic generation of high-frequency sine waves acting as the carrier signal.
* *AM & FM Modulation:* Real-time mathematical simulation of both Amplitude and Frequency modulation techniques.
* *Audio Integration:* Load, play, and process real audio files (.wav) as the modulating message signal.
* *Signal Visualization:* Clear graphical plots comparing the Message Signal, Carrier Signal, and the resulting Modulated Wave.
